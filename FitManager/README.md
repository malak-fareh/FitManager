# FitManager

## Structure du projet

FitManager/
├── model/ **Classes Java
├── dao/ **Accès base de données
├── servlet/ **Contrôleurs
├── util/ **Utilitaires (DBConnection)
├── webapp/ **Pages HTML, CSS, JS
├── database/init.sql -> (Base de données complète)



## Base de données

Le script SQL se trouve dans `database/init.sql`

Pour l'installer :
1. Ouvrez MySQL 
2. Exécutez le contenu de `database/init.sql`
3. Vérifiez que les tables sont créées
